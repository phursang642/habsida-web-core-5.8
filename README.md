# Services and Support Layout

## live demo [click here to ciew project] /(https://phursang642.github.io/habsida-web-core-5.8/)

This project is a responsive web layout built using HTML and CSS based on a Figma design.

## Features

- Responsive design (Mobile, Tablet, Desktop)
- Header with icons (menu, call, chat, profile, repair, status)
- Tabs section (Brands, Devices, Prices, etc.)
- Desktop layout with title on the left and action buttons on the right
- Clean UI based on Figma design
- BEM methodology for class naming
- Flexbox layout

##  Responsive Design

- Mobile: 320px
- Tablet: 768px
- Desktop: 1120px+

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
-  Fonts (Roboto)
